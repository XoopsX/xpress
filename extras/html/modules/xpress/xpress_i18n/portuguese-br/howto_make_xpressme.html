CONSTRUINDO O MÓDULO

  1. Crie uma pasta para o módulo (no exemplo utilizaremos xpress como nome).
  2. Copie os arquivos do XPressME Integration Kit para a pasta criada no 1º passo.
  3. Copie os arquivos do WordPress para a pasta criada no 1º passo.

LOCALIZAÇÃO

Se os arquivos de idioma necessários e o wp-config.php existirem na pasta extras/xpress_i18n, sobreescreva:
o arquivo wp-config.php;
a pasta language;
a pasta wp-content;
contido em extras/xpress_i18n/**/xpressme_modules_root na raiz do seu módulo.

INSTALAÇÃO DO MÓDULO XpressME

A partir do módulo criado em "CONSTRUINDO O MÓDULO", siga os seguintes passos para realizar a instalação:

  1. Faça upload, como qualquer outro módulo, para o diretório XOOPS_ROOT_PATH/modules/.
  2. Defina as permissões dos diretórios /templates e /wp-contents para 777.
  3. Instale o módulo.

DICAS

Desde que não coincida com o de nenhum outro do seu XOOPS, você pode escolher livremente o nome do diretório.
É possível duplicar o módulo apenas alterando seu nome (não D3 ready)
